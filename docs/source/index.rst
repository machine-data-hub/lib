Welcome to the Machine Data Hub's Documentation!
================================================

About
^^^^^
The Machine Data Hub python library is an open source project
for viewing, downloading, and suggesting datasets for Prognostics
Health Management Machine Learning Projects.

Eventually, the goal is to have machine-data-hub complete ETL
processes on datasets to allow users to download clean, easy to
use data.

Installation
^^^^^^^^^^^^
To install::

    pip install machine-data-hub
